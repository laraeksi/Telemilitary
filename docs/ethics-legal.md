# Ethical and Legal Considerations (CW1)

## Privacy and consent
- Telemetry is pseudonymous (no personal identifiers).
- Players are informed telemetry is collected for balancing and analytics.

## Data minimisation
- Only gameplay metrics (time, moves, resources) are captured.
- No real names, emails, or IP addresses are stored.

## Accessibility
- UI uses readable contrast and large buttons.
- Keyboard navigation is supported for core actions.

## IP and licensing
- Game assets are generated or locally created for the prototype.
- Dependencies are open-source and listed in `software-inventory.md`.
