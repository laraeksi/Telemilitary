# Optional game sounds

Drop OGG (or MP3) files here to use real sound effects instead of synthesized tones:

| File             | When it plays        |
|------------------|----------------------|
| `flip.ogg`        | Card flip            |
| `match.ogg`       | Matching pair        |
| `mismatch.ogg`    | Wrong pair           |
| `stage-complete.ogg` | Stage cleared     |
| `stage-fail.ogg`  | Out of time/moves    |
| `click.ogg`       | Button click (powerups, retry, quit) |
| `reward.ogg`      | Streak reward popup |
| `music.ogg`       | Background music (loops when Music is on) |

If a file is missing, the game falls back to Web Audio tones. No files are required.
